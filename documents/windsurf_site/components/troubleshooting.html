Troubleshooting

Encountering issues with Windsurf? Here are solutions for common problems:

Installation Issues

+
  • If installation fails, try downloading again or using an alternative method
  • Ensure your system meets the minimum requirements
  • Check for system permissions if the application won't install
  • If you get a "damaged file" error on macOS, try:
    xattr -d com.apple.quarantine /Applications/Windsurf.app

Performance Problems

+
  • Close unused applications to free up system resources
  • Restart Windsurf if it becomes sluggish
  • Check Task Manager/Activity Monitor for resource usage
  • Disable unused extensions that might be consuming resources
  • Try reducing the size of the workspace if it's very large

Cascade Not Responding

+
  • Verify your internet connection as Cascade requires online connectivity
  • Try switching to a different AI model
  • Restart the Cascade panel using the refresh button
  • Check your Windsurf subscription status and usage limits
  • Try clearing the Cascade conversation history

Import Issues

+
  • If VS Code settings don't import correctly, try importing specific settings manually
  • Some extensions may not be compatible with Windsurf
  • Check for any error messages during the import process
  • Try exporting settings from VS Code again with the latest version

MCP Connection Issues

+
  • Verify that MCP is enabled in settings
  • Check your internet connection
  • Ensure firewall settings aren't blocking MCP connections
  • Try restarting Windsurf to reset MCP connections

Diagnostic Tools

Windsurf includes several diagnostic tools to help identify and resolve issues:

System Information

Access via Help > System Info. Provides detailed information about your system configuration and Windsurf installation.

Developer Tools

Access via Help > Toggle Developer Tools. Provides access to console logs and network activity.

Reset Windsurf Settings

Access via File > Preferences > Reset Settings. Resets Windsurf to default settings if you encounter persistent issues.
