System Requirements

Before installing Windsurf, ensure your system meets these requirements:

For macOS

  • macOS 11 (Big Sur) or later
  • 8GB RAM minimum (16GB recommended)
  • 2GB free disk space
  • Intel or Apple Silicon processor

For Windows

  • Windows 10 64-bit or later
  • 8GB RAM minimum (16GB recommended)
  • 2GB free disk space
  • 64-bit processor

For Linux

  • Ubuntu 20.04 LTS or later (or compatible distributions)
  • 8GB RAM minimum (16GB recommended)
  • 2GB free disk space
  • 64-bit processor

Note: While Windsurf can run on systems with the minimum requirements, you'll have a better experience with the recommended specifications, especially when working with large projects or using Cascade extensively.
