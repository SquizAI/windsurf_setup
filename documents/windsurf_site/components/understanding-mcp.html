Understanding MCP

Model Context Protocol (MCP) extends Cascade's capabilities by connecting to external services and tools. Here's how to use it:

What is MCP?

MCP (Model Context Protocol) is a framework that allows Cascade to communicate with external services and access specialized functionality beyond its built-in capabilities. This enables Cascade to perform a wider range of tasks, from searching the web to interacting with APIs.

Enabling MCP

  1. Go to Windsurf Settings > Advanced Settings
  2. Scroll to the Cascade section
  3. Enable MCP integration

Available MCP Servers

Context7

Provides up-to-date documentation for libraries and frameworks

Example: "Get the latest React hooks documentation"

Firecrawl

Enables web search and content extraction capabilities

Example: "Search for examples of responsive layouts in CSS"

Stripe

Integrates with Stripe payment processing services

Example: "How do I implement Stripe checkout in my app"

Supabase

Connects to Supabase backend services

Example: "Show me how to set up authentication with Supabase"

Using MCP in Cascade

MCP capabilities are automatically available in your Cascade conversations. Here are some example uses:

Library Documentation

"Search for the latest React hooks documentation"

Cascade will use Context7 to retrieve up-to-date React hooks documentation.

Web Search

"Find examples of implementing Stripe payments in Node.js"

Cascade will use Firecrawl to search the web for relevant examples.

API Integration

"Get information about Supabase authentication methods"

Cascade will connect to Supabase MCP server to retrieve relevant information.

Setting Up Custom MCP Servers

Advanced users can set up custom MCP servers for specialized tasks. This requires configuring the server endpoint in Windsurf settings:

  1. Go to Windsurf Settings > Advanced Settings > MCP Configuration
  2. Click "Add Custom Server"
  3. Enter the server name and endpoint URL
  4. Configure any required authentication details
  5. Save your settings

Note: Custom MCP servers require proper security configurations. Only connect to trusted servers.
